\# Meeting Minutes Automation - MLOps Project



End-to-end MLOps system that converts multi-speaker meeting audio into structured meeting minutes:

transcripts, topics, action items, and executive summaries.



\## Project Structure

\- `data/` — raw and processed datasets (DVC tracked)

\- `src/` — core ML pipelines (ASR, topics, NER, summarization)

\- `app/` — FastAPI service

\- `notebooks/` — exploration \& demos

\- `models/` — trained artifacts (DVC tracked)

\- `docs/` — problem statement, acceptance criteria, reports

\- `tests/` — unit tests
